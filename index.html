Here is a command to replace all '>' characters to '>>' in a file FILE.TXT without opening VIM. 4) echo "Use TR to Lowercase a String" | tr "[A-Z]" "[a-z]" 5) uname -m The result for a 64-bit OS installation (for x86_64 architecture): x86_64 The result for a non-64-bit OS installation (for i686 architecture): i686 To know about the processor, run the following command: 6) lshw -class processor | grep width Shown below is the result for a 64-bit installation: width: 64 bits The result for a 32-bit installation: width: 32 bits 7) lsb_release -a No LSB modules are available.
